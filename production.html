WSU Magento Production Environment

How to Put into production

Generally if you are ready to push changes up to the server it will be done through the server administrator unless you are placing a new installation. There are going to be two types of updates, a new theme for your site, or a new extension. In the case of a theme you will use the FTP folder and sFTP transfer the theme up to your site folder. All new extension must be tested and installed by the WSU University Web Communications Team
